<checkbox-list> Demo

Simply import dist/index.html in your HTML file and use <checkbox-list> element.

Usage

Select bank account:
<checkbox-list>
  <label><input checkbox-list-all="true" type="checkbox">All</label>
  <label><input type="checkbox">Checking</label>
  <label><input type="checkbox">Saving</label>
</checkbox-list>
